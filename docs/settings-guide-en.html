1. What are Settings?

Settings allow you to customize how the application behaves according to your business needs and personal preferences.

Types of Settings

  • User Preferences - Your personal settings (language, date format)
  • Outlet Settings - Store/location-specific settings (tax rate, currency, POS settings)
  • Global Settings - Company-wide settings (company name, receipt footer)
📝 Important: Settings are organized into tabs based on their scope. You'll see only the settings relevant to your current role and permissions.

2. Accessing the Settings Menu

How to Access Settings:

  1. Open the application
  2. Tap the "Functions" menu
  3. Scroll down and select "Settings"
💡 Can't see the Settings menu? Contact your administrator to get the necessary permissions.

Settings Screen Layout

The Settings screen is organized into three tabs:

  • User Preferences - Your personal settings
  • Outlet Settings - Settings for the current store/location
  • Global Settings - Company-wide settings

3. Understanding Setting Scopes

Settings follow a 3-tier hierarchy that determines which value is used:

Scope Priority (Highest to Lowest):

  1. USER - Your personal preference (highest priority)
  2. OUTLET - Store/location setting
  3. TENANT - Company default (fallback)

How It Works

Example Scenario:

  • Company default language: English (TENANT scope)
  • Outlet 1 tax rate: 11% (OUTLET scope)
  • Your language preference: Indonesian (USER scope)

Result: You'll see the app in Indonesian (your USER setting overrides company default), but Outlet 1 will use 11% tax rate.

📝 Note: USER settings only affect you. OUTLET settings affect everyone using that outlet. TENANT settings affect the entire company.

5. Changing Settings

Step-by-Step Guide:

  1. Open the Settings screen
  2. Select the appropriate tab (User / Outlet / Global)
  3. Find the setting you want to change
  4. Tap on the setting to edit it
  5. Enter the new value
  6. Tap "Save Changes"

Real-Time Validation

The app validates your input as you type:

  • ✅ Valid input: Normal white input field
  • ❌ Invalid input: Red border with error message below
⚠️ Important: You cannot save changes if any setting has validation errors. Fix all errors before saving.

Batch Updates

You can change multiple settings at once:

  1. Make changes to multiple settings
  2. The "Save Changes" button will be enabled
  3. Tap "Save Changes" once to update all settings
💡 Efficiency Tip: Change all settings you need first, then save once. This is faster than saving after each change!

7. Read-Only Settings 🔒

Some settings are display-only and cannot be edited. These show important information about your account or device status.

Types of Read-Only Settings

Standard Read-Only Settings

Settings stored in the database that you can view but not edit.

Visual indicator: Lock icon 🔒 with greyed-out text

Device-Specific Settings ⭐

Settings that show information specific to your current device.

Example: Push notification registration status

Visual indicator: Modern pill/badge showing "Active" (green) or "Inactive" (red)

Current Read-Only Settings:

Push Notifications Registered ⭐
Shows if this device is registered to receive push notifications
💡 Why read-only? These settings are controlled by the system or device and should not be manually changed to ensure proper functionality.

8. Tips & Best Practices

🎯 Usage Tips

Check Before Saving Review all changes before tapping "Save Changes"
Use Default Values If unsure, use the default value provided by the system
Test Changes After changing settings, test the affected feature to ensure it works as expected
Document Changes Keep note of important setting changes for reference
Sync Regularly Open Settings screen periodically to sync latest changes from server

⚠️ Common Mistakes to Avoid

  • Ignoring validation errors - Always fix errors shown in red before saving
  • Changing wrong scope - Make sure you're editing in the correct tab (User/Outlet/Global)
  • Using extreme values - Stay within recommended ranges (e.g., don't set tax rate to 200%)
  • Forgetting to save - Changes aren't applied until you tap "Save Changes"

🔐 Security Tips

  • Don't share your user preferences with others - create separate accounts instead
  • Only change outlet settings if you're authorized to do so
  • Be careful with global settings - they affect everyone in the company
  • If unsure about a setting, ask your manager before changing it

9. Troubleshooting

❌ Can't Save Changes
Problem: The "Save Changes" button is disabled

Possible Causes:
  • There are validation errors (check for red borders)
  • You haven't made any changes yet
  • You don't have permission to change these settings
Solution:
  • Fix all validation errors shown in red
  • Make sure you've actually changed a value
  • Contact your administrator for permissions
⚠️ Changes Not Appearing
Problem: Saved changes but they're not showing in the app

Solution:
  1. Close and reopen the Settings screen to sync
  2. Restart the app if the setting isn't immediate-effect
  3. Check if you edited in the correct scope (User/Outlet/Global)
  4. Verify you have internet connection for sync
🔴 Validation Error Messages
Problem: Getting error messages when entering values

Common Errors:
  • "This field is required" - You must enter a value
  • "Value must be between X and Y" - Enter a number within the range
  • "Must be a valid number" - Enter numbers only, no letters
  • "Value must be one of: ..." - Choose from the allowed options
📱 Settings Not Loading
Problem: Settings screen shows loading or empty

Solution:
  1. Check your internet connection
  2. Wait a few seconds for sync to complete
  3. Close and reopen the Settings screen
  4. Restart the app
  5. Contact support if problem persists
🔄 Different Values on Different Devices
Problem: Settings show different values on another device

Explanation: This is normal for device-specific settings like push notification status

Solution:
  • Device-specific settings (marked with ⭐) are unique per device
  • User/Outlet/Global settings should sync across devices
  • If non-device settings differ, open Settings screen on both devices to force sync

❓ Frequently Asked Questions

Q: What's the difference between User, Outlet, and Global settings?

A: User settings are personal to you. Outlet settings affect everyone at that store location. Global settings affect the entire company.

Q: Can I reset a setting to its default value?

A: Currently, you can manually change it back to the default value shown in the setting description. Contact your administrator if you need help.

Q: Why can't I edit some settings?

A: Settings marked with 🔒 are read-only and managed by the system or administrators. You can view them but not change them.

Q: Do settings sync across my devices?

A: Yes! Most settings sync automatically. Device-specific settings (marked with ⭐) are unique to each device.

Q: What happens if I enter an invalid value?

A: The app will show a red border and error message. You must fix the error before you can save changes.

Q: Can I change multiple settings at once?

A: Yes! Change as many settings as you need, then tap "Save Changes" once to save all changes together.

Q: Why does changing language work immediately but other settings don't?

A: Some settings (marked with 🚀) are designed to take effect immediately. Others may require app restart or navigating to a different screen.

Q: Who can change outlet and global settings?

A: Only users with appropriate permissions can change these settings. Contact your manager or administrator for access.
